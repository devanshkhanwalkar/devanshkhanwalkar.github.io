What if the gem is sold out!

code goes here →

If unfortunately the gem is sold out we must not show the gem at all isn't it? how can we do that ?
well the answer is ng-hide. But before that just add a fiel in gem object named soldOut. This tells us whether the gem is in store or out of stock.
app.js

(function (){
var app=angular.module('store',[]);
app.controller('StoreController',function(){
this.product=gem;
});

var gem={
name:'Dodecahedron',
price: 2.95,
description: '....',
canPurchase: true,
soldOut:false,
}
})();

Working with Arrays

code goes here →

we now have 5 products but it is ridiculous to write html for each of 5 products.What if we have 5 more gem later? how do we display all products at once?
ng-repeat directive to the rescue!just put it on the div which you wanna repeat

index.html
<!DOCTYPE html >
<html ng-app="store">
<head >
<link href="bootstrap.min.css" rel="stylesheet"/>
<script src="angular.min.js"></script>
<script src="app.js"></script>
</head>
<body ng-controller="StoreController as store" >
<div class="container-fluid">
<div ng-repeat="product in store.products" ng-hide="product.soldOut">
<ul class="list-group">
<li class="list-group-item" >
<h1 class="text-primary">{{product.name}}</h1>
<h2><small>$ {{product.price}}</small></h2>
<h3><small>{{product.description}}</small></h3>
<button class="btn btn-primary" ng-show="product.canPurchase">Add to cart</button>
</div>
</li>
</ul>
</div>
</body >
</html >
